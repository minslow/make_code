HTLM에서 CSS언어 사용하는법

- <style> ~ </style>이라는 태그를 쓰면 안에 있는 것들을 CSS언어에 맞게 해석함.
- 또는 style="color:red"처럼 속성으로 사용 할 수 있다.

- a{ } 형식은 a태그에 해당하는 모든것을 의미한다.

-본격적으로 CSS코드를 알아보자!

글자 색 바꾸기

color:red;는 모든 a태그를 레드로 바꿈.

글씨 꾸미기(밑줄 등)

text-decoration: none; 은 밑줄을 없앰.
밑줄을 생기게 하려면 none 대신 underline 사용

글씨 사이즈 조정

글씨 사이즈를 바꾸려면 font-size:40px을 사용한다

글씨 중앙에 놓는법

text-align: center;과 같이 사용한다.
center 대신 left, right, justify등이 가능하다.

Class와 id에 관해서

태그에다가 class="saw"같은 형식을 집어넣으면 a{}을 밀치고 saw{}같은 형식을 쓸 수 있다.
class와 비슷한 id도 있는데 힘의 상관관계를 보자면 id>class> 태그 선택자 이다.
id는 #saw와 같이 사용하고 class는 .saw와 같이 사용한다.

박스

1. border 사용해서 재주껏
2. 테두리와 글자의 여백주는 방법 : padding:20px; 형식
3. 테두리와 테두리 사이의 간격 조정 방법 : margin:20[x; 형식]
4. display:block;같은 형식을 써서 블럭과 inline형식을 지정해 줄 수 있음
5. width로 테두리의 크기 결정 가능.

오직 디자인을 위한 태그

바로 div태그와 span태그이다. 설명을 조금 하자면 h1태그의 경우는 제목이라는 것을 담당하기때문에 무색무취인 div태그와 span태그의 필요성이 두드러진다.
div태그는 block element이고 span태그는 inline element이다.

화면 크기에 따라 달라지는 디자인

@media(max-width:800px){div{display:none;}} 이렇게 쓰면 화면이 800보다 작을 때 div태그안에 있는것들이 비활성화된다는 걸로 해석.

중복된 코드들 묶어버리기

<link rel="stylesheet" href="asdf.css">형식으로 사용하면 중복된 코드들을 한곳으로 묶을수있다.
